ข้อมูลหลักสูตร

ชื่อหลักสูตร : หลักสูตรวิศวกรรมศาสตรบัณฑิต สาขาวิชาวิศวกรรมสารสนเทศและเครือข่าย

รหัสหลักสูตร : 25620151100293

ระดับการศึกษา : ปริญญาตรี

คณะ / มหาวิทยาลัย : คณะเทคโนโลยีและการจัดการอุตสาหกรรม มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระนครเหนือ

คำอธิบายสั้น ๆ เกี่ยวกับหลักสูตร : เป็นหลักสูตรที่เน้นผลิตวิศวกรผู้เชี่ยวชาญด้านสารสนเทศและการสื่อสาร โดยมุ่งพัฒนาทักษะทั้งด้านซอฟต์แวร์ ฮาร์ดแวร์ และระบบเครือข่าย รวมถึงการประยุกต์ใช้เทคโนโลยีใหม่ๆ
